# Conference Tracker

## 상태 기준
- **Live on site**: 현재 사이트에 반영 완료
- **Live on site (needs cleanup)**: 페이지는 보이지만 자산명/문서/후속 점검이 남아 있음
- **Needs review**: 자료는 있으나 해석 또는 확인이 더 필요

## 현재 요약
- `Pages/Conference/` 아래 **1~26회 상세 페이지가 모두 존재**한다.
- `Pages/conferences.html`에도 **1~26회 목록이 모두 반영**되어 있다.
- 따라서 **13회는 더 이상 "다음 작업"이 아니며 이미 아카이브 반영 완료** 상태다.
- 현재 남은 일은 신규 생성보다 **문서 최신화, 자산 네이밍 정리, 일부 회차 후속 보강**에 가깝다.
